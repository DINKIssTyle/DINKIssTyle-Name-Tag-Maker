# 🏷️ DINKIssTyle 명찰 제작기

대량의 명찰(네임태그)을 손쉽게 디자인하고, PDF로 출력할 수 있는 데스크탑 애플리케이션입니다.

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **📐 용지 / 태그 설정** | A4, Letter 등 프리셋 선택 또는 커스텀 용지 크기 지정. 태그 크기·배열·여백·간격을 자유롭게 조정 |
| **🎨 텍스트 박스** | 여러 개의 텍스트 박스를 추가하고, 폰트·크기·색상·정렬·글자 간격 등 세부 속성 편집 |
| **📋 스프레드시트** | 엑셀/구글 시트에서 복사(Ctrl+V)하여 대량 데이터 한 번에 붙여넣기 가능. CSV/TSV 가져오기 지원 |
| **🖼️ 배경 이미지** | 명찰에 배경 이미지 삽입 |
| **👁️ 미리보기** | 실시간 다중 페이지 미리보기 |
| **📄 PDF 출력** | 지정된 폰트가 임베딩된 고품질 PDF 생성. 텍스트 자동 축소(Auto-scaling) 지원 |
| **💾 프로젝트 저장/불러오기** | `.ntag` 파일로 프로젝트 저장 및 불러오기. 덮어쓰기 저장(Ctrl+S) 지원 |
| **🖨️ 바로 인쇄** | PDF 생성 후 시스템 기본 앱으로 인쇄 |

## 🖥️ 스크린샷

> 앱을 실행하면 좌측 설정 패널과 우측 캔버스로 구성된 메인 화면이 표시됩니다.

## 📦 설치

### 요구 사항

- **Python** 3.12 이상
- **uv** (권장) 또는 pip

### uv 사용 (권장)

```bash
# 저장소 클론
git clone https://github.com/dinki/DINKIssTyle-Name-Tag-Maker.git
cd DINKIssTyle-Name-Tag-Maker

# 의존성 설치 및 실행
uv run main.py
```

### pip 사용

```bash
pip install -r requirements.txt
python main.py
```

### 의존성 패키지

| 패키지 | 용도 |
|--------|------|
| `PyQt6` | GUI 프레임워크 |
| `Pillow` | 미리보기 이미지 렌더링 |
| `reportlab` | PDF 생성 |

## 🚀 사용 방법

### 1. 용지 및 태그 설정
좌측 패널에서 용지 크기, 태그 크기, 배열(열×행), 여백, 간격을 설정합니다. 설정 변경은 실시간으로 캔버스에 반영됩니다.

### 2. 텍스트 박스 추가
`+ 추가` 버튼으로 텍스트 박스를 추가합니다. 이름, 소속, 직책 등 필요한 만큼 추가할 수 있습니다. 각 텍스트 박스는 캔버스에서 드래그하여 위치와 크기를 조절할 수 있습니다.

### 3. 데이터 입력
`📋 스프레드시트` 버튼을 눌러 데이터를 입력합니다. 엑셀에서 Ctrl+C → 스프레드시트에서 Ctrl+V로 대량 붙여넣기가 가능합니다.

### 4. 미리보기 및 출력
- `👁 미리보기`로 결과물을 확인합니다.
- `PDF 저장`으로 PDF 파일을 저장하거나, `🖨 인쇄`로 바로 인쇄합니다.

## ⌨️ 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+O` | 프로젝트 열기 |
| `Ctrl+S` | 저장 (덮어쓰기) |
| `Ctrl+Shift+S` | 다른 이름으로 저장 |

## 🔤 폰트 지원

- macOS, Windows, Linux 각 OS의 시스템 폰트 디렉토리를 자동 탐색합니다.
- 한국어 폰트명(예: `맑은 고딕`, `궁서`, `배달의민족 주아`)과 영문 파일명 사이의 자동 변환을 지원합니다.
- **PDF 출력 시**, 텍스트가 지정된 텍스트 박스 크기를 초과하면 자동으로 폰트 크기가 축소됩니다.
- `.ttf` / `.otf` 폰트가 ReportLab에서 가장 잘 지원됩니다. `.ttc` 파일은 일부 호환성 제한이 있을 수 있습니다.
